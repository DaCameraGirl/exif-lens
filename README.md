# EXIF Lens – by DaCameraGirl 📸

> EXIF metadata viewer for photographers.

Drag a photo in → see every camera setting, instantly.

**Live:** https://dacameragirl.github.io/exif-lens/

![Next.js](https://img.shields.io/badge/Next.js-15-black)
![License](https://img.shields.io/badge/license-MIT-blue)

---

## Features

- 🎯 **Drag & drop** – or paste from clipboard
- 📷 **Full EXIF readout** – camera body, lens, aperture, shutter, ISO, focal length, flash, white balance, metering, and more
- 🗺️ **GPS map** – see exactly where the photo was taken (OpenStreetMap)
- 🧹 **Strip EXIF** – one-click clean download, removes all metadata before posting online
- 📦 **Batch support** – drop multiple photos, flip through them like a filmstrip
- 📋 **Copy JSON** – grab the raw EXIF data
- 🎨 **Beautiful UI** – dark, photographer-focused, fast

Supported formats: JPEG, HEIC, PNG, TIFF, WebP, AVIF

---

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **exifr** – EXIF parsing
- **Leaflet / react-leaflet** – maps
- **Lucide** – icons
